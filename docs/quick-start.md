# TabView プラグイン クイックスタートガイド

**所要時間**: 約5分

---

## 1. プラグインのインストール

### ステップ1: ZIPファイルの準備

プラグインZIPファイルの場所:
```
C:\projects\kintone-plugins\plugins\tab-view\plugin.zip
```

### ステップ2: kintoneにインストール

1. kintoneにログイン
2. 右上の⚙️アイコン → **kintone システム管理**
3. **プラグイン** → **読み込む**
4. `plugin.zip`を選択してアップロード
5. ✅ インストール完了！

---

## 2. テスト用アプリの作成

### フィールド構成例

| フィールドコード | フィールド名 | フィールドタイプ |
|----------------|------------|----------------|
| `name` | 氏名 | テキスト（1行） |
| `email` | メールアドレス | テキスト（1行） |
| `phone` | 電話番号 | テキスト（1行） |
| `department` | 部署 | ドロップダウン |
| `address` | 住所 | テキスト（複数行） |
| `company` | 会社名 | テキスト（1行） |
| `position` | 役職 | テキスト（1行） |
| `hire_date` | 入社日 | 日付 |

---

## 3. プラグインをアプリに追加

1. 作成したアプリを開く
2. ⚙️ → **アプリの設定**
3. **設定** → **プラグイン**
4. **TabView - タブビュー**を追加
5. **アプリを更新**

---

## 4. タブの設定

### ステップ1: 設定画面を開く

1. プラグイン一覧で**TabView**の⚙️アイコンをクリック

### ステップ2: タブを追加

#### タブ1: 基本情報
1. **+ タブを追加**をクリック
2. タブ名: `基本情報`
3. アイコン: `👤`
4. フィールド: ✅ 氏名, ✅ メールアドレス, ✅ 電話番号, ✅ 部署
5. **タブを保存**

#### タブ2: 詳細情報
1. **+ タブを追加**をクリック
2. タブ名: `詳細情報`
3. アイコン: `ℹ️`
4. フィールド: ✅ 住所, ✅ 会社名, ✅ 役職, ✅ 入社日
5. **タブを保存**

### ステップ3: 設定を保存

1. **設定を保存**ボタンをクリック
2. ✅ 「設定を保存しました」と表示される

---

## 5. 動作確認

### ステップ1: レコードを作成

1. アプリを開く
2. **+**ボタンでレコードを追加
3. すべてのフィールドに値を入力
4. **保存**

### ステップ2: タブUIを確認

1. 作成したレコードの詳細画面を開く
2. 画面上部に**タブUI**が表示される:
   ```
   ┏━━━━━━━━━┓ ┌─────────┐
   ┃ 👤 基本情報 ┃ │ ℹ️ 詳細情報 │
   ┗━━━━━━━━━┛ └─────────┘
   ```
3. 「基本情報」タブのフィールド（氏名、メール、電話番号、部署）が表示される

### ステップ3: タブを切り替え

1. **ℹ️ 詳細情報**タブをクリック
2. フィールドが切り替わる（住所、会社名、役職、入社日が表示される）

## トラブルシューティング

### タブが表示されない

**原因**: タブが1個しか設定されていない、またはタブが未設定

**解決策**: 設定画面で2個以上のタブを作成してください

### フィールドが表示されない

**原因**: タブにフィールドが割り当てられていない

**解決策**: 設定画面でタブを編集し、フィールドを選択してください

### 設定が保存できない

**原因**: タブ名が未入力、またはフィールド未選択

**解決策**: エラーメッセージを確認し、必要な項目を入力してください
