## デザインコンセプト

### キーワード
- **シンプル**: 余計な装飾を排除
- **直感的**: 説明なしで使える
- **モダン**: 2026年の標準デザインに準拠
- **kintoneとの調和**: 標準UIと違和感なく共存

### デザイン原則
1. **視認性**: タブが見やすく、選択状態が明確
2. **操作性**: クリックしやすいサイズ、ホバー時のフィードバック
3. **一貫性**: 色、サイズ、余白のルールを統一
4. **パフォーマンス**: アニメーションは最小限、動作は高速

## アクセシビリティ

### 1. キーボード操作

- **Tab**: 次のタブに移動
- **Shift + Tab**: 前のタブに移動
- **Enter / Space**: タブを選択
- **矢印キー（←→）**: タブ間を移動

### 2. スクリーンリーダー対応

```html
<button
  role="tab"
  aria-selected="true"
  aria-controls="tab-panel-1"
  id="tab-1"
>
  基本情報
</button>

<div
  role="tabpanel"
  aria-labelledby="tab-1"
  id="tab-panel-1"
>
  <!-- タブコンテンツ -->
</div>
```

### 3. カラーコントラスト

- **テキストと背景**: WCAG AA基準（4.5:1以上）を満たす
- **選択状態**: 色だけでなく、下線でも表現

## 実装時の技術詳細

### 1. HTMLコンポーネント構造

```tsx
// React + TypeScript
interface TabProps {
  tabs: Array<{
    id: string;
    label: string;
    icon?: string;
    fields: string[];
  }>;
  activeTab: string;
  onTabChange: (tabId: string) => void;
}

const TabView: React.FC<TabProps> = ({ tabs, activeTab, onTabChange }) => {
  return (
    <div className="tab-container">
      {/* タブヘッダー */}
      <div className="tab-header" role="tablist">
        {tabs.map(tab => (
          <button
            key={tab.id}
            role="tab"
            aria-selected={activeTab === tab.id}
            className={`tab-button ${activeTab === tab.id ? 'active' : ''}`}
            onClick={() => onTabChange(tab.id)}
          >
            {tab.icon && <span className="tab-icon">{tab.icon}</span>}
            <span>{tab.label}</span>
          </button>
        ))}
      </div>

      {/* タブコンテンツ */}
      <div className="tab-content-wrapper">
        {tabs.map(tab => (
          <div
            key={tab.id}
            role="tabpanel"
            className={`tab-content ${activeTab === tab.id ? 'active' : ''}`}
            hidden={activeTab !== tab.id}
          >
            {/* フィールドを表示 */}
          </div>
        ))}
      </div>
    </div>
  );
};
```
